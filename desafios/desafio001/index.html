Evolução do HTML

A evolução do HTML começou com a criação de Tim Berners-Lee em 1990, evoluindo do HTML 1.0 (básico) para versões mais complexas como HTML 2.0 (1995) e HTML 4.01 (1999), que introduziram tabelas, formulários e melhorias de padrão. A mudança mais significativa foi a chegada do HTML5 em 2014, que revolucionou a web com elementos multimídia nativos, tags semânticas e APIs para aplicações mais ricas e interativas, além de ter impulsionado a separação entre estrutura (HTML), estilo (CSS) e interatividade (JavaScript).
